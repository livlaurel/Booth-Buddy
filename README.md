Booth Buddy is a web-based photo booth application that lets users capture and customize digital photo strips, inspired by traditional film booths. With features like filters, stickers, and easy sharing, Booth Buddy brings nostalgic photo-booth fun into a modern, accessible, and social-media-friendly experience.
